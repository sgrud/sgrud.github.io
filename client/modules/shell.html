Attribute

  • Attribute(name?: string): (component: Component, propertyKey: PropertyKey) => void
    • Defined in packages/shell/src/component/attribute.ts:41

    Prototype property decorator factory. Applying this decorator to a property of a registered Component binds the decorated property to the corresponding attribute of the custom component and adds the attribute to the observedAttributes array. If no name parameter is supplied, the name of the decorated property will be used instead. If both, a parameter initializer and an initial attribute value are supplied, the attribute value takes precedence.

    example

    Decorate a property.

    import { Attribute, Component } from '@sgrud/shell';

    declare global {
      interface HTMLElementTagNameMap {
        'example-component': ExampleComponent;
      }
    }

    @Component('example-component')
    export class ExampleComponent extends HTMLElement implements Component {

      @Attribute()
      public field?: string;

      public get template(): JSX.Element {
        return <span>Attribute value: {this.field}</span>;
      }

    }

Component

  • Component<S, R>(selector: S, inherits?: R): <T>(constructor: T) => T
    • Defined in packages/shell/src/component/component.ts:132

    Class decorator factory. Registers the decorated class as component with the CustomElementRegistry. This decorator should be used in conjunction with the Attribute prototype property decorator and will re-render, when a bound property changes. While any custom component which is registered by this decorator inherits basic rendering functionality, any overridden method will cancel out its super logic. This ensures for a minimal and transparent abstraction of the functionality the CustomElementRegistry provides, while allowing for full customization.

    example

    Register a component.

    import { Component } from '@sgrud/shell';

    declare global {
      interface HTMLElementTagNameMap {
        'example-component': ExampleComponent;
      }
    }

    @Component('example-component')
    export class ExampleComponent extends HTMLElement implements Component {

      public readonly styles: string[] = [`
        span {
          font-style: italic;
        }
      `];

      public get template(): JSX.Element {
        return <span>Example component</span>;
      }

    }

Reference

  • Reference(ref: Key, observe?: keyof HTMLElementEventMap[]): (component: Component, propertyKey: PropertyKey) => void
    • Defined in packages/shell/src/component/reference.ts:43

    Prototype property decorator factory. Applying this decorator to a property of a registered Component while supplying the reference key and, optionally, an array of events to observe, will replace the decorated property with a getter returning the referenced node, once rendered. If an array of events is supplied, whenever one of those events is emitted by the referenced node, the referenceChangedCallback is called with the reference key, the referenced node and the emitted event.

    example

    Reference a node.

    import { Component, Reference } from '@sgrud/shell';

    declare global {
      interface HTMLElementTagNameMap {
        'example-component': ExampleComponent;
      }
    }

    @Component('example-component')
    export class ExampleComponent extends HTMLElement implements Component {

      @Reference('example-key')
      private readonly span?: HTMLSpanElement;

      public get template(): JSX.Element {
        return <span key="example-key"></span>;
      }

    }

  • createElement(type: Function | keyof HTMLElementTagNameMap, props?: Record<string, any>, ref?: Key): Element
    • Defined in packages/shell/src/component/runtime.ts:90

    JSX element factory. Provides jsx-runtime-compliant bindings to the incremental-dom library. This factory function is meant to be implicitly imported by the transpiler and returns an array of bound incremental-dom function calls, representing the created JSX element. This array of bound functions can be rendered into an element attached to the DOM through the render function.
