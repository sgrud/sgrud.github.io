Interface Module

Interface describing the shape of a module. This interface is aligned with some package.json fields. It further specifies an optional sgrudDependencies mapping, as well as an optional webDependencies mapping, which both are used by the Kernel to determine SGRUD module dependencies and runtime (web) dependencies.

example

An example module definition.

import type { Kernel } from '@sgrud/core';

const module: Kernel.Module = {
  name: 'module',
  version: '0.0.0',
  exports: './module.exports.js',
  unpkg: './module.unpkg.js',
  sgrudDependencies: {
    sgrudDependency: '^0.0.1'
  },
  webDependencies: {
    webDependency: {
      exports: {
        webDependency: './webDependency.exports.js'
      },
      unpkg: [
        './webDependency.unpkg.js'
      ]
    }
  }
};
