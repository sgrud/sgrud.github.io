Class RouterLink

Custom component extending the HTMLAnchorElement. This component provides a declarative way to invoke the Router, while maintaining compatibility with SSR/SEO aspects of SPAs. This is achieved by rewriting absolute hrefs to be contained within the Router.baseHref and replacing the default browser behavior when onclicked with Router.navigate.

example

A router-link.

<a href="/example" is="router-link">Example</a>
