Class RouterTask Abstract

Abstract base class to implement Router.Tasks. By Targeting or otherwise providing an implementation of this abstract base class to the Linker, the implemented handle method is called whenever a new Router.State is triggered by navigating. This interceptor-pattern makes complex routing strategies like asynchronous module-retrieval and the like easy to be implemented.

example

Simple router task stub.

import { Provider, Target } from '@sgrud/core';
import type { Router, RouterTask } from '@sgrud/shell';
import type { Observable } from 'rxjs';

@Target<typeof ExampleRouterTask>()
export class ExampleRouterTask
  extends Provider<typeof RouterTask>('sgrud.shell.router.RouterTask') {

  public override handle(
    prev: Router.State,
    next: Router.State,
    handler: Router.Task
  ): Observable<Router.State> {
    throw new Error('Stub!');
  }

}
