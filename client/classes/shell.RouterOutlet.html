Class RouterOutlet

Custom element extending the HTMLSlotElement. When this component is constructed, it binds the Router to itself while supplying the value of its baseHref attribute as Router.baseHref and the presence of a hashBased attribute on itself as Router.hashBased. This component should only be used once, as it will be used by the Router as Router.outlet to render the current Router.State.

example

A router-outlet.

<slot baseHref="/example" is="router-outlet">Loading...</slot>
