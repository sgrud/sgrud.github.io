Class Registry<K, V>

The Registry is a Singleton map used by the Provider to lookup Provided constructors by magic strings upon class extension. Magic strings should represent sane package paths in dot-notation. To programmatically provide constructors by magic strings to extending classes, the inherited MapConstructor or Map.prototype.set methods are available. The former will insert all entries into this singleton Registry map, internally calling the latter for each.

decorator

Singleton

example

Preemptively provide a constructor by magic string.

import type { Registration } from '@sgrud/core';
import { Registry } from '@sgrud/core';
import { Service } from './service';

new Registry<Registration, Service>([
  ['sgrud.example.Service', Service]
]);
